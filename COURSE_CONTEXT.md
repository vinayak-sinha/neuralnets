# ATCS Neural Networks Course Context

This repository comes from Vinayak Sinha's work in **ATCS Neural Networks** at The Harker School in San Jose, California, during the 2025-2026 school year.

The course syllabus describes a first-principles path through artificial neural networks:

- Parallel distributed processing and why neural networks are useful classifiers.
- Perceptrons, linearly separable problems, and the XOR limitation.
- Nonlinear classifiers and iterative supervised learning.
- Widrow-Hoff learning, gradient descent, and backpropagation.
- General fully connected feed-forward networks.
- Binary data representations for inputs, truth tables, and learned weights.
- BMP image processing, pixel extraction, normalization, and image pattern recognition.

The repository mirrors that sequence. The early control files exercise small Boolean networks; the generalized network code supports configurable layer layouts; the image utilities convert and normalize BMP data; and the packed image datasets plus saved weights support a final image-recognition task.

Public Harker course context also describes this class as having students spend the semester creating a basic multilayer perceptron, with the goal of understanding pattern classifiers at the lowest levels rather than treating deep learning as a black box.
